# Training Planner (План тренировок)

## Автор
Долбилова Василиса Алексеевна
## Описание
Графическое приложение (GUI) для планирования и учета тренировок. 
Приложение позволяет добавлять записи о тренировках, просматривать их в виде таблицы, 
фильтровать по типу активности и дате, а также сохранять данные в формате JSON.

## Технологии
- Python 3.x
- Tkinter (для GUI)
- JSON (для хранения данных)
- Git (для контроля версий)

## Функционал
1. **Добавление тренировки**:
   - Ввод даты (формат ДД.ММ.ГГГГ).
   - Выбор типа тренировки (Бег, Плавание, Силовая и т.д.).
   - Указание длительности в минутах.
2. **Валидация данных**:
   - Проверка формата даты.
   - Проверка, что длительность — положительное число.
3. **Просмотр и фильтрация**:
   - Таблица всех добавленных тренировок.
   - Фильтр по типу тренировки.
   - Фильтр по конкретной дате.
4. **Сохранение данных**:
   - Автоматическое сохранение в файл `workouts.json` при добавлении новой записи.
   - Загрузка данных из файла при запуске приложения.
